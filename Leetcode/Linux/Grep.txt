Grep: Used to search specific phrase, words or patterns inside text files and show matching lines on screen.

commands useful
1. grep "python" file.txt - searches for python in file.txt
2. grep -i "python" file.txt - case insensitive search 
3. grep -c "python" file.txt - count no of lines
4. grep -l "Unix" * - matching filename
5. grep -w "unix" file.txt - check whole words
6. grep -o "unix" file.txt - display matched patterns/string.
7. grep -n "unix" file.txt- displays line no
8. grep -v "unix" file.txt - string/pattern which does not match.
9. grep "^unix" file.txt - This can be used in grep to match the lines which start with the given string or pattern. 
10. grep "os$" file.txt - This can be used in grep to match the lines which end with the given string or pattern.
11. grep -f pattern.txt geekfile.txt - to read patterns from file
12. grep -e "agarwal" file.txt - used for specific expressions.
13. grep -A1 learn file.txt - prints n specific lines from file

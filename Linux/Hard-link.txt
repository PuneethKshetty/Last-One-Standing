Hard Links:
1. Here hard links will have same inode number as original file.
2. Hardlinks are not allowed for directories. [Only a superuser* can do it].
3. It cannot be used across file systems.
4. Data present in original file will still be available in hardlinks.
5. If original file is removed, link will still work as it accesses the data the original was having access to.
6. Hard links are faster than soft links.

ln t1.txt hl [commands]

ls -il t1.txt sl hl[to check inode numbers]
